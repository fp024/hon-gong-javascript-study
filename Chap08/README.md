## 08-2 예외 처리와 고급

#### 예외 객체

  | 속성 이름 | 설명        |
  | --------- | ----------- |
  | name      | 예외 이름   |
  | message   | 예외 메시지 |

#### 예외 강제 발생

```javascript

// 조금더 자세하게 예외를 발생
```
