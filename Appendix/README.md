# 구 버전의 인터넥 익스플로러 지원



2년 전만해도 호환성 유지를 위해 IE 지원을 해줘야했을 것 같은데... 

지금은 Cromium 기반의 MS Edge가 나왔기 때문에, IE 11도 지원 안하는 사이트들이 늘어날 것 같다.

부록에서 저자님은, 최신 자바스크립트 코드를 IE에서 실행하기 위해 바벨과 es-hims에 대해 언급 하셨다.



### 바벨 

문법적인 변환

* 테스트 사이트
  * https://babeljs.io/repl



## es-shims
